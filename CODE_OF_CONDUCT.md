# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) verbatim.

The full text is published at the link above and is incorporated here by reference.

## Short version

- Be kind. Assume good faith.
- No harassment, discrimination, or personal attacks.
- Disagreement is fine; contempt is not.
- Maintainers will enforce this consistently and privately when needed.

## Reporting

If you experience or witness behaviour that violates the Contributor Covenant, please email the maintainer privately. Reports are handled confidentially.

You should get an acknowledgement within 72 hours and a resolution decision within 14 days.

## Scope

This applies in all project spaces — issues, pull requests, discussions, commit messages, code review — and when representing the project in public.

## Enforcement

Maintainers follow the [Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines): correction, warning, temporary ban, permanent ban — graduated to fit the situation.
